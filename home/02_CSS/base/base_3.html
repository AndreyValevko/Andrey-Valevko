Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным .2em, пр помощи свойства margin-bottom

Статья 2

Размер блока
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным .2em, пр помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внитри данной статьи для этой картинки.

Нижний блок, подвал. Занимет всю доступную ширину. Высота определяется помещённый вовнутрь контентом
